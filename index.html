JSX
It is a JavaScript syntax extension that allows you to use HTML-like syntax to describe the structure of an interface.
Without JSX:

							const App = () => {
								const box =	React.createElement("div");
								return box.innerHTML = 'Hello World';
							}
						

							const App = () => {
								const box =	React.createElement("div");
								const title =	React.createElement("title", null, "Hello world!");
								return box.append(title);
							}
